## 📝 Description

python-projects is a repository containing a collection of independent Python scripts designed to automate everyday digital tasks and manage basic workflows. It provides functional scripts targeting real-world operations such as Excel sheet automation, email dispatch, local file management, and quick utility generation.

## ✨ Key Features

- **📊 Excel Spreadsheet Automation** — Process and manipulate tabular data in Excel spreadsheets using automated Python scripts like XL_automation.py.
- **✉️ Automated Email Messaging** — Send and automate email notifications using email_ai.py and local email automation scripts.
- **📁 Directory File Organization** — Sort and organize cluttered system directories automatically using the file organizer utility.
- **🌦️ Weather Data Scraping** — Fetch and parse real-time weather information from web resources using the weather.py script.
- **🎓 Student Records Management** — Manage, track, and store basic student information locally through a dedicated terminal-based system.
- **🔑 Password and QR Generation** — Generate secure randomized passwords and create custom QR codes using standalone utility scripts.

## 🎯 Use Cases

- Automating the generation and updating of Excel monthly reports from raw data files.
- Cleaning up cluttered directories on local machines by organizing files automatically into categorized folders.
- Retrieving local weather reports using terminal-based web scraping techniques.
- Setting up local automated email systems for notifications or quick file sharing.
- Creating quick utility tokens, such as secure passwords and scannable QR codes, directly from the terminal.
